# CRBC Import Matchs

Plugin WordPress pour importer les matchs du CRBC66 depuis des fichiers Excel (.xlsx).

## Fonctionnalités

- Import du calendrier (matchs à venir)
- Import des résultats (matchs joués avec scores)
- Création automatique des équipes externes manquantes
- Création automatique des termes de taxonomie manquants
- Détection des doublons
- Interface d'administration en français
